📦 StockFlow – Inventory Management System
A modern, cloud-based inventory management system built using HTML, CSS, and JavaScript, with real-time backend powered by Firebase and version control via GitHub.
________________________________________
🚀 Features
•	📊 Real-time inventory tracking
•	➕ Add, ✏️ edit, and 🗑️ delete products
•	🔍 Search and filter functionality
•	📈 Live statistics (total products, stock value, low stock alerts)
•	⚡ Responsive and modern UI
•	☁️ Cloud-based data storage using Firebase
________________________________________
🛠️ Tech Stack
Frontend:
•	HTML5
•	CSS3
•	JavaScript
Backend:
•	Firebase Firestore (real-time database)
Hosting:
•	Netlify
Version Control:
•	GitHub
________________________________________
☁️ Cloud Architecture
•	Firebase → Backend (PaaS)
•	Netlify → Frontend Hosting (SaaS)
•	GitHub → Version Control

⚙️ How It Works
1.	User opens the web application
2.	Firebase initializes and connects to Firestore
3.	Inventory data is fetched and displayed
4.	User performs actions:
o	Add product
o	Edit product
o	Delete product
5.	Data updates in Firestore
6.	Real-time updates reflect instantly on UI
